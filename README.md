# Управление
| Клавиша | Действие |
|-------------|-------------|
| → | Сдвинуть вправо |
| ← | Сдвинуть влево |
| ↓ | Сдвинуть вниз |
| Q | Поврот по часовой стрелке |
| W | Поворот против часовой стрелки |
| space | drop фигуры |
